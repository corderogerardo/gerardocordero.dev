## About This Repository

This repo is the source of [**gerardocordero.dev**](https://gerardocordero.dev) — an **editorial-style native portfolio** built the way I build production apps. It's a pnpm + Turborepo monorepo; the live app is **Expo / React Native** with expo-router and NativeWind.

```
apps/portfolio   →  the live app (Expo, expo-router, NativeWind)  ← active
packages/ui      →  shared React Native UI library
apps/old-web     →  legacy CRA portfolio (archived)
```

**The loop** — the three commands that say a change is correct:

```bash
pnpm typecheck   # tsc --noEmit
pnpm lint        # eslint (eslint-config-expo)
pnpm test        # jest via jest-expo (unit)
```

Beyond the loop: **Maestro** end-to-end flows (iOS + Android), **EAS Workflows** for build / e2e / deploy, GitHub Actions CI, and a SemVer release pipeline with in-app version display. The same care for testing, code reviews, and developer experience I bring to a team — applied to my own portfolio.
